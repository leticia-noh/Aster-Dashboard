# Dashboard Aster - Frontend
A dashboard da Aster é uma aplicação voltada para o gerenciamento e visualização de métricas estratégicas relacionadas aos serviços de distribuição de softwares de criação.

## Instalação
#### Instale o Node (caso ainda não tenha):
Confira: <https://www.nodejs.org>


#### Instale as dependências:  
``npm install``


#### Para rodar a aplicação execute:  
``npm run dev``

> Acesse em ``http://localhost:3000``  
> **Nota:** parte das funcionalidades requer o servidor em execução
